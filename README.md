# My Campus - Student Support App
**University of Salford | Industrial Project**

## 📱 Project Overview
My Campus is a mobile application designed to improve student-teacher interaction and gamify attendance tracking at the University of Salford. It features a minimalist aesthetic focused on utility and accessibility.

## ✨ Key Features
- **Smart Dashboard:** Real-time class locations and assignment countdowns.
- **Interactive Map:** Targeted navigation for Salford campus facilities.
- **Attendance Tracking:** Gamified progress bars to encourage engagement.
- **Student Settings:** Personalized notification toggles and profile management.

## 🛠️ Tech Stack
- **Framework:** React Native (Expo)
- **Design System:** Custom Indigo/Blue palette (derived from legacy CSS)
- **Version Control:** Git & GitHub

## 🚀 Getting Started
1. **Clone the repo:** `git clone https://github.com/YOUR_USERNAME/my-campus-app.git`
2. **Install dependencies:** `npm install`
3. **Run the app:** `npx expo start`
4. **View on Mobile:** Download the 'Expo Go' app and scan the QR code in your terminal.

## 📁 Project Structure
- `/screens`: Contains the core UI for Login, Dashboard, and Settings.
- `/constants`: Style guides including colors and spacing.
